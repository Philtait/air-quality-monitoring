# Air Quality Monitoring & Analysis Project

## Project Overview
This project analyzes air quality sensor data (PM2.5) across multiple U.S. cities to identify patterns, trends, and insights about air pollution.

## Objectives
- Understand air quality sensors and their parameters
- Perform Exploratory Data Analysis (EDA) 
- Visualize air quality trends over time and geography

##  Technologies Used
- **Python 3.x** - Programming language
- **Pandas** - Data manipulation
- **NumPy** - Numerical computing
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter Notebooks** - Interactive analysis
